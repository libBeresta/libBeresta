## Сборка библиотеки

Для сборки используется система [CMake](https://cmake.org).

```bash
$ md build && cd build
$ cmake ..
$ make
```

Библиотека содержит некоторое количество конфигурационных опций, их можно посмотреть, выполнив команду 

```bash
$ cmake -LAH | grep BRST
```

Часть опций только обозначена, но пока не реализована.

Отметим реализованные опции.

### LIBBRST_EXAMPLES[=OFF]
Указывает, собирать ли примеры вместе со сборкой библиотеки
По умолчанию отключена.

### LIBBRST_SHARED_LIB[=ON]
Указывает, собирать ли библиотеку в виде shared object (dynamic library) или статически.
По умолчанию собирается динамическая библиотека.
