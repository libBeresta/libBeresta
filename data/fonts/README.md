## Шрифты демо примеров

Для текстовых примеров используется комплект [шрифтов PT Public Pack](https://www.paratype.ru/collections/pt/44157).
Необходимо самостоятельно скачать архив в эту папку и развернуть его:

```sh
$ unzip PT-Public-Pack.ottt.zip
```

При необходимости можно исправить пути до TTF-файлов в текстах примеров.
