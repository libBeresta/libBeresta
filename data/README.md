# Вспомогательные файлы библиотеки libBeresta

В этой папке собраны вспомогательные файлы и скрипты, предназначенные для
работы с исходным кодом, библиотекой в целом или сборочным конвейером.

* `clang-format`
   Конфигурационный файл для Clang Format. Используется для содержимого
   каталога [src](../src), для заголовочных файлов не используется.
* `page-sizes`
   Скрипт и данные для генерации файла с размерами бумаги, поддерживаемыми
   библиотекой.
* `spell`
   Конфигурационные файлы и скрипт проверки орфографии исходных текстов
   библиотеки.
